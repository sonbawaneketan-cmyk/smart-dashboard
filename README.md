# 📊 Smart Data Cleaning & Visualization Dashboard

## 🚀 Overview
The Smart Data Dashboard is a Streamlit-based web application designed to clean, analyze, and visualize data in an interactive way. It allows users to upload datasets, manually enter data, or provide raw text input, and instantly generate insights with charts and summaries.

---

## 🎯 Features
- 📂 Upload CSV files
- ✍️ Manual data entry
- 🧾 Convert raw text into structured data
- 🧹 Automatic data cleaning
- 📊 Interactive visualizations (Bar Chart & Pie Chart)
- 🔍 Data filtering by category
- 📌 Summary insights (Total Amount & Top Category)
- 📥 Download professional HTML reports

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
